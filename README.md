# Todo List на Python

Консольное приложение для управления списком дел.

## Возможности

- Добавление, удаление, просмотр задач
- Отметка о выполнении
- Сохранение в JSON-файл
- Защита от некорректного ввода

## Как запустить

### Вариант 1: Запуск из исходного кода

```bash
python todo.py
```

### Вариант 2: Готовый .exe файл

В папке dist находится todo.exe — запустите двойным щелчком.
